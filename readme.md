# Settings
```bash
# 가상환경 생성
python3 -m venv .venv

# 가상환경 접근 (Linux & macOS)
source .venv/bin/activate

# 가상환경 접근 (Windows)
.venv\Scripts\activate

# 가상환경 비활성화
deactivate

# 파이썬 라이브러리 설치
pip install -r requirements.txt
```
